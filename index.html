Coord Input▶
While LINE active, type to open input.
Values entered in current unit (mm):
100,50 — absolute X,Y
@50,-20 — relative dX,dY
@100<45 — polar dist<angle°
50 or @50 — length only; extends toward the cursor, or the locked angle if angle lock is on
<45 then 100 — set an exact angle, then a length on the next line
@<30 then 100 — turn 30° from the current heading, then a length
LINE 0,0 0,10 COMMIT — a whole script line, space-separated; each token runs like its own Enter
🎤 Voice: tap the mic — "at fifty comma twenty five", "circle", "delete", "commit".
🎤 Or a whole sentence: "draft a line starting at zero zero ending at zero ten" runs all three steps. Unrecognized speech is quietly ignored, never guessed at.
